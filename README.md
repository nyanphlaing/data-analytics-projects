# Data Analytics Portfolio

Personal data analytics portfolio built using Python and SQL.
Each project covers a different industry and business problem.

## Projects

### 1. E-Commerce Sales Analysis

Analysis of 541,909 transactions from a UK retailer identifying revenue trends,
peak trading periods, and customer concentration risk.
**Tools:** Python, pandas, Matplotlib, Seaborn

### 2. Video Game Sales Analysis

Global video game sales analysis across platforms, genres, and publishers from
1990 to 2016, with Python and SQL used side by side.
**Tools:** Python, pandas, Matplotlib, Seaborn, SQL

### 3. Netflix Content Analysis

Analysis of Netflix's 8,800+ title library identifying content strategy patterns
by country, genre trends, and growth trajectory.
**Tools:** Python, pandas, Matplotlib, Seaborn, SQL

### 4. Superstore Sales Analysis

Business performance analysis identifying $135,000 in recoverable profit through
discount policy changes, with regional and segment recommendations.
**Tools:** Python, pandas, Matplotlib, Seaborn, SQL, CASE WHEN

## Skills Demonstrated

- Data cleaning and preparation
- Exploratory Data Analysis (EDA)
- Data visualization
- SQL querying (GROUP BY, WHERE, JOINS, CASE WHEN, subqueries)
- Business insight and recommendation writing
